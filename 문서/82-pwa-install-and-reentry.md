# PWA Install And Reentry

## 목적

모바일 웹은 설치형 앱보다 첫 진입이 쉽지만, 재방문 습관이 약해지기 쉽다. 이 문서는 PWA 설치와 재진입 UX를 정의한다.

## 핵심 장치

- 첫 사용 후 2회차 방문에서만 설치 제안
- 설치 후 첫 재진입 시 최근 대화와 초안을 바로 복원
- 홈 화면 아이콘, 앱 제목, 테마색 일관화

## 원칙

- 설치 권유는 강요하지 않는다.
- 설치 여부와 상관없이 동일한 핵심 경험을 보장한다.
- 설치 이점은 `더 빨라진다`가 아니라 `더 끊기지 않는다`로 설명한다.

## 성공 기준

- 설치 프롬프트가 방해가 아니라 복귀 장치로 인식된다.
