# Desktop Layout Recipes

## 목적

데스크톱의 우위는 기능 수가 아니라 `업무 상황별 배치 조합`에서 나온다. 이 문서는 대표적인 창/패널 레이아웃 레시피를 정리한다.

## 기본 레시피

- 기본 3열
- 목록+대화 집중
- 대화+컨텍스트 레일
- 다중 창 분리
- 세로 모니터 요약형

## 시나리오별 추천

- 회의 중: 대화+파일 패널
- 운영 대응: 메인 목록+긴급 방 팝아웃
- 글 작성: 대화 본문 확장형

## 성공 기준

- 사용자는 자기 업무 스타일에 맞는 기본 배치를 저장하고 다시 불러올 수 있다.
