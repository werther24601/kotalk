# KoTalk Master Plan

이 폴더는 KoTalk의 마스터 기획 세트다.

현재 문서 세트는 루트 기획 문서와 세부 UX 아틀라스로 구성되며, 제품 전략, 사용자 경험, 실제 사용자 리뷰, 릴리즈 운영 기준, 신뢰/보안 표면까지 함께 다룬다.

## 고정 기준

- 제품 목표는 `국내 사용자가 즉시 익숙하게 쓰는 한국어 메신저`다.
- 핵심은 복제가 아니라 `짧은 복귀, 낮은 피로도, 높은 밀도`다.
- 공개 가입 정책은 `초대코드 중심`보다 `이메일/휴대폰 기반 1회성 인증`을 우선한다.
- 디자인 기준은 `플랫`, `각진 구조`, `화이트 원톤`, `텍스트 최소화`, `멀티 윈도우 친화성`이다.
- 장기 방향은 `오픈소스 코어 + 운영 서비스 + 자체 호스팅/규제 환경 대응`의 병행이다.
