# User Interview Script And Study Guide

## 목적

문서 확장만으로는 실제 사용성을 확인할 수 없다. 이 문서는 인터뷰와 관찰 세션을 반복하기 위한 질문 가이드를 정리한다.

## 핵심 질문

- 첫 화면에서 무엇이라고 느꼈는가
- 어디서 망설였는가
- 어떤 순간에 다시 쓰고 싶다고 느꼈는가
- 어떤 순간에 아직 프로토타입 같다고 느꼈는가

## 관찰 포인트

- 입력 전 망설임 시간
- 대화 찾기 시간
- 실패 메시지 반응
- 하단 바 이해도

## 성공 기준

- 매 빌드마다 최소 3명 이상에게 같은 흐름을 반복 측정한다.
