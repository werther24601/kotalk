# UX Expansion Atlas

이 디렉터리는 기존 문서 루트 마스터 플랜을 세부 주제로 3배 이상 확장하기 위한 아틀라스다. 현재 이 하위 아틀라스만 `253개 문서`로 구성된다.

원칙:
- 루트 문서 README는 방향과 우선순위를 요약한다.
- 문서 atlas는 세부 UX, 운영, 리뷰, 릴리즈 설계를 깊게 분해한다.
- 실제 산출물에 대한 비판적 견해와 사용자 관점 리뷰를 숨기지 않는다.
- 실제로 사용해 본 뒤 불편했던 점을 범주별 QA 문서에 다시 기록한다.
