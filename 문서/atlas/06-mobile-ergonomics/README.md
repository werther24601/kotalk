# 모바일 인체공학과 저피로 UI

이 묶음은 모바일 인체공학과 저피로 UI 관점에서 사용자 체감 간편함을 세부 주제로 분해한 아틀라스다.

## 범위
- 주요 목적: 작은 화면에서도 짧고 덜 피로한 동작을 만드는 구조
- 우선 사용자: 모바일 우선 사용자
- 우선 채널: Mobile Web + Android

## 현재 산출물에 대한 비판적 요약
- 최근 개편으로 목적지 구분은 좋아졌지만 목록 상단 밀도와 첫 대화방의 빈 느낌은 아직 강하다.
- 작은 화면에서 행동보다 설명이 앞서는 순간이 남아 있어 첫 신뢰를 갉아먹을 수 있다.
