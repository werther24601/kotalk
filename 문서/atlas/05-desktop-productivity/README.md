# 데스크톱 생산성 표면

이 묶음은 데스크톱 생산성 표면 관점에서 사용자 체감 간편함을 세부 주제로 분해한 아틀라스다.

## 범위
- 주요 목적: Windows에서 다중 창과 키보드 흐름으로 체감 시간을 줄이는 구조
- 우선 사용자: 업무형 파워유저
- 우선 채널: Windows

## 현재 산출물에 대한 비판적 요약
- 현재 데스크톱은 화이트 컴팩트 톤은 잡혔지만 다중 창과 키보드 우선 흐름은 아직 초입이다.
- 업무형 사용자 기준으로는 창 기억, 팝아웃, 검색 오버레이가 더 있어야 대체재처럼 느껴진다.
