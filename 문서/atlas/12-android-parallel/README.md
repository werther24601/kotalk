# Android 병렬 채널

이 묶음은 Android 병렬 채널 관점에서 사용자 체감 간편함을 세부 주제로 분해한 아틀라스다.

## 범위
- 주요 목적: 웹과 독립적으로 Android 채널을 설계하고 병렬 배포하는 구조
- 우선 사용자: 모바일 일상 사용자 + 업무형 사용자
- 우선 채널: Android

## 현재 산출물에 대한 비판적 요약
- Android는 전략과 릴리즈 표면 문서만 있고 실제 사용자 체감은 아직 검증되지 않았다.
- 모바일 웹을 그대로 옮기면 안 되고, Material과 알림, 권한, 백그라운드 제약에 맞춘 재설계가 필요하다.
