# Bookmarks, Tags, And Collections

## 목적

메신저 안에 쌓이는 유용한 메시지를 다시 쓰려면 북마크와 가벼운 분류 체계가 필요하다. 이 문서는 보관과 태그 전략을 정리한다.

## 사용자 문제

- 중요한 메시지가 채팅 흐름 속에 묻힌다.
- 북마크만으로는 맥락 분류가 부족하다.

## 핵심 장치

- 북마크
- 태그
- 컬렉션
- 최근 저장

## 원칙

- 태그는 사용자가 직접 만들 수 있지만, 기본 세트는 소수로 시작한다.
- 컬렉션은 개인 저장 중심이며 팀 협업 기능과는 분리한다.

## 성공 기준

- 메시지를 저장한 뒤 다시 찾는 시간이 대폭 줄어든다.
