# Shared Links And Reference Panels

## 목적

업무 대화에서 링크는 기억보다 재발견이 중요하다. 이 문서는 공유 링크를 따로 모아 읽는 참조 패널 UX를 정의한다.

## 사용자 문제

- 링크가 메시지 본문에 묻혀 나중에 찾기 어렵다.
- 링크 미리보기가 없어 다시 열기 전에 판단이 어렵다.

## 핵심 기능

- 링크 패널
- 제목/도메인/보낸 사람 요약
- 최근/중요/핀된 링크 정렬
- 읽지 않은 링크 표시

## 성공 기준

- 회의 링크, 문서 링크, 참고 자료를 방별로 빠르게 다시 찾는다.
